Detailed Description

A generic implementation of a quadrature encoder that tracks the position of the encoder in counts and the speed the encoder is traveling at in counts per second.

Parameters
timerThe timer the encoder channels are on.
timing_timerThe timer that is used to track time in order to calculate speed accurately.
Attention
The timing_timer must be configures such that it counts values in microseconds
Parameters
prev_countThe previous count timer read.
curr_countThe current count timer reads.
prev_timeThe previous count timing_timer read.
curr_timeThe current count timing_timer reads.
posThe position of the encoder in counts.
speedThe speed the motor is traveling at in counts per second
dxThe difference in position between the previous encoder reading and the current encoder reading in counts
dxThe difference in time between the previous encoder reading and the current encoder reading in microseconds
